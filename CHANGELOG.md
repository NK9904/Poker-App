## Release Notes

### Version 2.0.0

This major release introduces significant improvements to the AI analysis engine, enhanced 3D
visualizations, and a complete UI redesign. The application now provides more accurate GTO
calculations and better user experience.

### Version 1.5.0

Focus on advanced analysis features including range vs range analysis and tournament considerations.
Improved performance and user interface enhancements.

### Version 1.4.0

Added essential poker tools like pot odds calculator and advanced position analysis. Enhanced AI
model performance and 3D visualizations.

### Version 1.3.0

Introduced real-time opponent modeling and advanced GTO strategies. Improved accuracy and user
interface.

### Version 1.2.0

Enhanced range calculator and hand evaluation capabilities. Performance optimizations and bug fixes.

### Version 1.1.0

Added support for additional poker variants and improved mobile experience. Updated dependencies and
documentation.

### Version 1.0.0

Initial release with core poker analysis functionality, AI integration, and 3D visualization
features.
